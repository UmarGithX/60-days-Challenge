## 🛠 Recommended Learning Plan

### Phase 1 — ML Foundation (Weeks 1–3)
- **Topics:** ML basics, Scikit-learn, NumPy, Pandas (advanced), model evaluation
- **Resources:**
  - [fast.ai Practical Deep Learning](https://course.fast.ai/) — free, hands-on
  - [Kaggle Learn — ML courses](https://www.kaggle.com/learn) — free, bite-sized
  - [Scikit-learn official docs](https://scikit-learn.org/)
- **Goal:** Train and evaluate your first ML model locally

### Phase 2 — LLMs & APIs (Weeks 4–6)
- **Topics:** OpenAI API, Anthropic API, LangChain, Prompt Engineering
- **Resources:**
  - [DeepLearning.AI Short Courses](https://www.deeplearning.ai/short-courses/) — free, world-class
  - [OpenAI Cookbook](https://cookbook.openai.com/)
  - [LangChain Documentation](https://docs.langchain.com/)
- **Goal:** Build a functional AI chatbot with memory and tool use

### Phase 3 — RAG & Agents (Weeks 7–9)
- **Topics:** Embeddings, Pinecone / ChromaDB, RAG pipelines, AI Agents
- **Resources:**
  - [LlamaIndex Documentation](https://docs.llamaindex.ai/)
  - [Pinecone Learning Center](https://www.pinecone.io/learn/)
  - Hugging Face tutorials on YouTube
- **Goal:** Build a document Q&A system using RAG

### Phase 4 — Deployment & MLOps (Weeks 10–12)
- **Topics:** FastAPI, Docker basics, Hugging Face Spaces, CI/CD for ML
- **Resources:**
  - [FastAPI Official Docs](https://fastapi.tiangolo.com/)
  - [Hugging Face Spaces](https://huggingface.co/spaces)
  - Docker Getting Started guide
- **Goal:** Deploy a production-ready AI API
